## Ringkasan Folder

- `Module 1/Capstone 1`: aplikasi terminal Rental Mobil (modular, MySQL, statistik, visualisasi).
- `Module 1/Session 2`: latihan dasar Python.
- `Module 1/Session 3`: latihan lanjutan.
- `Module 1/Session 4`: latihan session 4.
- `Module 1/Session 6`: latihan session 6.

## Menjalankan File Latihan

Contoh jalankan salah satu file latihan dari root project:

```powershell
python "Module 1/Session 2/exercise1.py"
```

## Menjalankan Capstone 1

Masuk ke folder capstone 1 lalu liat README.md jalankan aplikasi:

```powershell
cd "Module 1/Capstone 1"
python app.py
```

Catatan:

- Detail setup dependency Capstone 1 ada di `Module 1/Capstone 1/README.md`.
- Disarankan menggunakan virtual environment (`.venv`) untuk menjalankan project.
